About This Website

This is an R Markdown website, created by Jeff Goldsmith and Vadim Zipunnikov for the JSM 2019 continuing education course titled “Functional Data Analysis for Wearables: Methods and Applications”.

In the short course, we’ll actively use the 'tidyverse and refund packages, and sometimes will use refund.shiny and tidyfun. Code to install the packages follows, and version information about each of these packages and their dependencies is at the bottom of this page.

install.packages(c("refund", "refund.shiny", "tidyverse"))
devtools::install_github("tidyfun/tidyfun")
